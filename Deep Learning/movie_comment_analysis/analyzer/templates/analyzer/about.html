🔬 ML Pipeline
  • 1

    Data Loading

    The IMDb CSV is auto-detected in the project root. It contains 50,000 labeled movie reviews (25K positive, 25K negative).

  • 2

    Text Preprocessing

    Each review is cleaned: HTML tags stripped, text lowercased, punctuation removed, and English stopwords filtered out.

  • 3

    TF-IDF Vectorisation

    Text is converted to numerical features using TF-IDF with unigrams + bigrams (1,2), up to 50,000 features, with sublinear TF scaling.

  • 4

    Logistic Regression

    A Logistic Regression model (C=1.0, max_iter=1000) is trained on the vectorised features. LR is well-suited for high-dimensional sparse text data.

  • 5

    Evaluation

    The model is evaluated on the held-out 20% test set. Metrics include Accuracy, Precision, Recall, F1-Score, Confusion Matrix, and Classification Report.

  • 6

    Serialisation

    The fitted vectorizer is saved as vectorizer.pkl and the model as sentiment_model.pkl for fast inference.

  • 7

    Django Web App

    Artefacts are loaded once at startup. The web UI accepts raw review text, cleans it, vectorizes it, and returns the label + confidence score in real-time.
